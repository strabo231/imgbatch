# Contributing to ImgBatch

Thanks for your interest in improving ImgBatch!

## Reporting Bugs
- Include OS and ImageMagick version
- Image format and size
- Command used and error message

## Suggesting Features
- Describe the image processing operation
- Use case explanation
- Example input/output

## Pull Requests
1. Fork repository
2. Create feature branch
3. Test with sample images (not personal photos!)
4. Follow code style
5. Update documentation

## Testing
Test with sample images, not sensitive or personal photos.
